# Modelo de Inteligencia Artificial

El modelo entrenado para la clasificación de residuos no se encuentra almacenado en este repositorio debido a su tamaño.

---

## 📦 Descarga del modelo

Puedes descargar el modelo desde el siguiente enlace de Google Drive:

👉 https://drive.google.com/file/d/1br48ms-wZiWBcbBR9LvP-QogoGz7_OOv/view?usp=sharing

---

## 📁 Ubicación en el proyecto

Una vez descargado, colócalo en:

backend/modelo_residuos.h5

---

## ⚙️ Uso en el sistema

El backend de FastAPI carga el modelo automáticamente desde esta ruta para realizar la clasificación de residuos en tiempo real.
